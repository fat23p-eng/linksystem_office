# Link Hub — วิธีใช้งาน

## ไฟล์ที่ได้รับ
- `index.html` — หน้าเว็บหลัก (แก้ไขแล้ว)
- `Code.gs` — ไม่ต้องแก้ไข (เหมือนเดิม)

---

## สิ่งที่เปลี่ยนแปลง

| ฟีเจอร์ | เดิม | ใหม่ |
|---|---|---|
| หน้าแรก | ต้อง Login ก่อน | **เปิดดูได้ทันที ไม่ต้อง Login** |
| แสดงรายการ | Card grid | **ตารางแยกหมวดหมู่** |
| เพิ่มรายการ | Admin เท่านั้น | **ทุกคนทำได้** |
| แก้ไขรายการ | Admin เท่านั้น | **ทุกคนทำได้** |
| ลบรายการ | Admin เท่านั้น | **Admin เท่านั้น (เหมือนเดิม)** |
| ปุ่ม Admin | ต้องใส่รหัสผ่านที่ login screen | **ปุ่ม "Admin" มุมขวาของ navbar** |

---

## วิธีติดตั้ง

1. **Google Apps Script (Code.gs)**
   - เปิด Google Sheet → Extensions → Apps Script
   - วางโค้ด Code.gs → Deploy → Web App (Execute as Me, Anyone)
   - คัดลอก URL ที่ได้

2. **index.html**
   - เปิดไฟล์ แก้บรรทัดนี้:
     ```
     const API_URL = 'https://script.google.com/macros/s/ใส่_URL_ตรงนี้/exec';
     ```
   - วาง URL ที่ได้จากขั้นตอนที่ 1

## การใช้งาน

### ผู้ใช้ทั่วไป (ไม่ต้อง Login)
- ดูรายการทั้งหมดในรูปแบบตาราง
- ค้นหาได้ กรองหมวดหมู่ได้
- กด "เพิ่มรายการ" หรือ ✏️ เพื่อเพิ่ม/แก้ไขได้ (ไม่ต้องรหัสผ่าน)

### Admin
- กดปุ่ม **Admin** มุมขวาของ navbar → ใส่รหัสผ่าน (default: `admin1234`)
- จะเห็นปุ่ม 🗑 ในแต่ละแถว (ลบรายการ)
- เห็นปุ่ม **นำเข้า CSV**, **ส่งออก**, **ตั้งค่า** เพิ่มขึ้นมา
- กด "ออกจาก Admin" เพื่อกลับโหมดทั่วไป

### เปลี่ยนรหัส Admin
- Login Admin → กดปุ่ม "ตั้งค่า" → กรอกรหัสใหม่ในช่อง "รหัส Admin ใหม่" → บันทึก
